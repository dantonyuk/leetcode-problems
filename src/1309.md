## Solution

```scala
object Solution {
  def freqAlphabets(s: String): String = {
    val num = """(\d(?:\d(?=#))?)#?""".r
    num.replaceAllIn(s, { case num(x) => (x.toInt + 96).toChar.toString })
  }
}
```
